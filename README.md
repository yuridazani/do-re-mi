## ğŸš€ Getting Started

### Prerequisites

Ensure you have **Node.js** (v18+) installed on your machine.

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/yuridazani/do-re-mi.git
    cd do-re-mi
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```
